# 1.4 — Coordinate System

## 大局观 (High-Level Map)

游戏里的每一个物件都有一个位置——角色在哪里、子弹飞到哪里、敌人站在哪里。

要描述位置，我们需要一个**坐标系（Coordinate System）**。

你在数学课上学过坐标系，但 Phaser 的坐标系和数学课里的**不一样**。这是初学者最常搞混的地方之一，所以我们要仔细搞清楚。

## 1. 复习：数学里的 2D Cartesian Plane

在数学里，**2D Cartesian Plane（二维笛卡尔平面）** 是这样的：

```
         Y ↑
         |
         |
─────────+─────────→ X
         |
         |
       (0,0) 在中间
```

- **原点 (0, 0)** 在画面**中间**
- **X 轴** 向右增大，向左减小
- **Y 轴** 向上增大，向下减小

---

## 2. Phaser 的坐标系：原点在左上角

Phaser 的坐标系和数学里的不同，它更接近"屏幕坐标"：

```
(0,0) ──────────────→ X 增大
  |
  |
  |
  ↓
  Y 增大
```

关键区别：

| | 数学坐标系 | Phaser 坐标系 |
|---|---|---|
| 原点位置 | 画面中央 | **左上角** |
| Y 轴方向 | 向上增大 | **向下增大** |
| X 轴方向 | 向右增大 | 向右增大（相同） |

**Origin（原点）** 是坐标系中 (0, 0) 所在的点。在 Phaser 里，原点固定在屏幕的**左上角**。

---

## 3. X 轴和 Y 轴的方向

### X 轴：向右增大

```
(0, 0)    (100, 0)    (200, 0)    (300, 0)
  ●──────────●──────────●──────────●
  └─────────────────────────────────→ X 增大（向右）
```

X 值越大，物件越靠**右**。X 值越小，物件越靠**左**。

### Y 轴：向下增大

```
(0, 0)   ●
          |
(0, 100) ●
          |
(0, 200) ●
          |
          ↓ Y 增大（向下）
```

Y 值越大，物件越靠**下**。Y 值越小，物件越靠**上**。

**这是和数学最大的不同：在 Phaser 里，Y 向下是增大，不是减小。**

---

## 4. 单位：Pixel

Phaser 坐标的单位是 **Pixel（像素）**。

**Pixel** 是屏幕上最小的显示单位。一个 800 × 600 的游戏画布，意思是：
- 宽度方向有 800 个像素（X 从 0 到 799）
- 高度方向有 600 个像素（Y 从 0 到 599）

坐标 `(400, 300)` 就是水平方向第 400 像素、垂直方向第 300 像素的位置——也就是这个画布的**正中间**。

---

## 5. 实际例子

假设游戏画布是 800 × 600：

```
(0,0)─────────────────────(800,0)
  |                           |
  |     (400, 300)            |
  |         ●                 |
  |      中心点               |
  |                           |
(0,600)────────────────(800,600)
```

| 坐标 | 在画布上的位置 |
|------|----------------|
| (0, 0) | 左上角 |
| (800, 0) | 右上角 |
| (0, 600) | 左下角 |
| (800, 600) | 右下角 |
| (400, 300) | 正中间 |

### 常见直觉错误

> "我把 Y 设大一点，角色应该往上移"

**错！** 在 Phaser 里，Y 增大 = 往下移。如果你想让角色往上移，要**减小** Y 值。

## Self-Check

**Q1.** Phaser 坐标系的原点 (0, 0) 在哪里？

> **答：** 在屏幕（画布）的**左上角**。

---

**Q2.** 在 Phaser 里，如果你把一个物件的 Y 值从 100 增加到 200，这个物件会向哪个方向移动？

> **答：** 向**下**移动。Phaser 的 Y 轴方向是向下增大的。

---

**Q3.** 一个 600 × 400 的游戏画布，正中间的坐标是什么？

> **答：** (300, 200)。X 中点是 600 ÷ 2 = 300，Y 中点是 400 ÷ 2 = 200。

---

**Q4.** Phaser 坐标和数学坐标的最大区别是什么？

> **答：** 数学坐标的 Y 轴向上增大，Phaser 坐标的 Y 轴向下增大。另外原点位置也不同：数学坐标原点在中心，Phaser 原点在左上角。

---

**Q5.** 坐标的单位是什么？一个 800 × 600 的画布，X 的范围是多少？

> **答：** 单位是 Pixel（像素）。800 × 600 的画布，X 的范围是 0 到 799（共 800 个像素）。
